# esra abidi — Portfolio

موقع شخصي (Portfolio) لعرض المشاريع والمهارات، مبني بـ HTML, CSS, JavaScript فقط — بدون أي framework، سهل النشر على GitHub Pages.

## التخصيص
- بدّل الاسم والروابط (GitHub, LinkedIn, email) في `index.html`.
- عدّل المشاريع من المصفوفة `PROJECTS` في `script.js`.
- ضع صورة CV باسم `cv.pdf` داخل مجلد `assets/`.

## النشر على GitHub Pages
1. ارفع الملفات لمستودع جديد على GitHub.
2. Settings → Pages → Branch: main → Save.
3. الموقع يصبح متاح على: `https://USERNAME.github.io/REPO-NAME/`
